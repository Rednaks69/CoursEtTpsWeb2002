FlexBox

Le module de mise en page de boîte flexible, facilite la conception d'une structure de mise en page flexible et réactive sans utiliser de flottement ou de positionnement. Afin de manager les position du contenus de container Flexbox offre plusieurs propriétés

Les propriétés de Flex container sont :

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
